# Veil Protocol

Skill for clawd — install and your agent instantly understands the Veil network.

## Install

Download `veil-protocol.skill` and install via clawd.

## What Your Agent Learns

- What Veil Protocol is and how it works
- How to provide idle AI capacity and earn USDC
- How to access AI models privately with crypto
- How to contribute code via RBOB (Rule-Based Open Build)
- Token economics, security model, privacy architecture

## Quick Start

```bash
# Provide
$ clawd provide start

# Consume
$ clawd credits add 10

# Build
$ clawd build
```
